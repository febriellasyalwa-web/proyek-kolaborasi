My Experience Text

Last month, I created a website project for a school assignment. At first, I was very excited because I wanted to make a cool and attractive website. I started by designing the layout, writing HTML and CSS code, and adding some features using JavaScript.

However, during the process, I faced many challenges. One of the biggest problems was code errors. Sometimes the website did not appear because there were mistakes in the HTML structure or missing brackets in the CSS and JavaScript code. I also once wrote the wrong file name, so the CSS was not connected and the website layout became messy.

In addition, some features did not work as I expected. For example, the navigation bar could not be clicked, images did not appear, and the responsive layout looked untidy on smaller screens. I had to check the code line by line to find the mistakes.

Even though I often felt tired and struggled during the process, I did not give up. I tried to find solutions on the internet, watched tutorials, and fixed the code step by step. After several revisions and improvements, the website was finally completed successfully.

From this experience, I learned to be more patient, careful, and persistent when facing problems while creating a website.
